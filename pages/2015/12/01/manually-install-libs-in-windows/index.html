Manually Install libs in Windows

Tue 01 December 2015

手动安装python组件

linux下面比较简单，但是有时候要在windows下面干点事情。比如说，在windows下面安装中文分词软件jieba

  1. 下载jieba，并解压到d盘，比如说 D:\jieba-0.37\
  2. windows下面打开cmd，进入命令行模式
  3. 输入 cd D:\jieba-0.37 进入文件夹
  4. 输入 dir 查看是不是有setup.py，确认有
  5. 最后输入 python setup.py install 安装
