variable selection in linear regression

  • decision tree to pick top predictable factors
  • single factor analysis: by rsquared_adj
  • this is to mimic the SAS forward variable selection
  • best subset regression
  • one variable transformation for all variables
## 1. decision tree to pick top predictable factors this is to run the regression decision tree first, then get the feature importance. The __feature importances__. The higher, the more important the feature. The importance of a feature _is computed as the (normalized) total reduction of the criterion_ brought by that feature. It is also known as the Gini importance. For regression, it is the mean square error. wzxhzdk:0
## 2. single factor analysis: by rsquared_adj wzxhzdk:1
## 3. this is to mimic the SAS forward variable selection each step, the variable is selected by their f-test value. wzxhzdk:2
## 4. best subset regression it will loop through all the variables combination of the Xs. So this will be very time consuming: in you have n variables, then you will run `n!` regressions. wzxhzdk:3
## 5. one variable transformation for all variables if you have 10 variables, and each variable has different transformation with the same starting prefix name. then you can force one of these transformed variable in the model. The reason to include one kind of transform is to avoid multicollinearity. if you include gdp and ln(gdp) then it is very likely they are high correlated. wzxhzdk:4
